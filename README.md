# it-learning 
# 最近1年で自分で調べて試したIT関連のこと

## 1. SNS活動を始めたきっかけ

私は以前、アイドルを目指して活動しており、自分をより多くの人に知ってもらうために、TikTokや配信アプリなどを使ったSNS活動に力を入れていました。

活動を続ける中で、「どうすれば動画を最後まで見てもらえるのか」「どんな投稿が反応を得やすいのか」に興味を持ち、自分で調べながら改善を繰り返していました。

---

## 2. 実際に行った分析・改善

具体的には、投稿時間を変えて再生数や反応を比較したり、動画冒頭の数秒の構成を変えて視聴維持率を確認したりしていました。

また、タイトルや概要欄、ハッシュタグによって反応がどのように変化するのかも分析していました。特に、最初の数秒で視聴者が離脱しやすいことに気づき、動画の入り方やテロップの見せ方を工夫するようになりました。

さらに、他の人気投稿も参考にしながら、「どのような要素が多くの人の興味を引くのか」を比較して研究していました。

---

## 3. ChatGPTなどのITツール活用

ChatGPTも活用し、動画タイトル案や投稿ネタ、概要欄、キャッチコピーなどを考える際の補助として使っていました。

自分だけでは思いつかない視点や表現を得ることができ、投稿内容を考える時間の短縮にもつながりました。また、複数案を比較しながら改善できたことで、効率よく投稿準備を進められるようになりました。

---

## 4. 学んだこと・効果

最初は感覚的に投稿していましたが、数字や反応を見ながら改善を重ねることで、「どのような工夫が結果につながるのか」を意識するようになりました。

その結果、以前より反応が伸びる投稿が増え、分析しながら改善する習慣も身につきました。

この経験を通して、ITツールやAIを活用しながら、自分で課題を見つけて試行錯誤することの大切さを学びました。また、データをもとに改善を続ける姿勢は、今後もさまざまな場面で活かしていきたいと考えています。
